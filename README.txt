ปรับ Doubly Linked List Insert:
1. newNode.next ชี้ไปโหนดถัดไป
2. newNode.previous ชี้ไปโหนดก่อนหน้า
3. next ของโหนดก่อนหน้า ชี้มาที่ NewNode
4. previous ของโหนดถัดไปเดิม ชี้มาที่ NewNode
5. Insert Complete
